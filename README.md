# 🛠️ Digitools - The Ultimate Digital Tool Marketplace

Digitools is a cutting-edge e-commerce platform built with **React** and **Tailwind CSS**. It provides a sleek, high-performance interface for users to discover and purchase premium digital tools, AI assistants, and design assets.

## 🔍 Project Overview
**Digitools** is designed to be a one-stop-shop for digital creators. It features a modern landing page with sections for featured products, business statistics, pricing plans, and a step-by-step guide on how to get started. The application is fully responsive and optimized for a fast, intuitive user experience.

| Category | Details |
| :--- | :--- |
| **Project Name** | Digitools |
| **Framework** | React 18 (Vite) |
| **Styling** | Tailwind CSS + DaisyUI |
| **Routing** | React Router 7 |
| **State** | React Hooks (useState) |
| **Icons** | Lucide React |

## ✨ Key Features

### 🛒 1. Dynamic Shopping Experience
- **Interactive Cart**: Add and remove items instantly with real-time state updates.
- **Cart Persistence**: Navigation-friendly cart that maintains items across different pages.
- **Visual Feedback**: Integrated `React Toastify` for beautiful, non-intrusive notifications.

### 🏠 2. Premium Landing Page
- **Hero Banner**: Engaging first impression with a clear Call-to-Action.
- **Stats Dashboard**: Showcases project growth (50K+ Users, 200+ Tools, 4.9 Rating).
- **Process Steps**: A clear guide for users to understand the platform workflow.

### 💰 3. Professional Pricing & Listing
- **Dynamic Pricing Cards**: Clean cards showing features, price, and billing periods.
- **Tagging System**: Highlighting "Popular", "Best Seller", and "New" products.
- **Detailed Product Info**: Every tool comes with a feature list and specific descriptions.

## 🚀 Tech Stack

### Frontend
- **React.js**: Core library for building the UI.
- **Vite**: Ultra-fast build tool and development server.
- **React Router DOM**: Client-side routing for multi-page feel.

### Styling & UI
- **Tailwind CSS**: Utility-first CSS for custom designs.
- **DaisyUI**: Component library for consistent and accessible UI elements.
- **Lucide React**: Premium icon set for modern aesthetics.

### Utilities
- **React Toastify**: Toasts for better user interaction feedback.

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. **Clone the repo**
   ```bash
   git clone https://github.com/yourusername/digitools.git
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Start development server**
   ```bash
   npm run dev
   ```
